# История обновлений плагина RengaBri4ka

# Версия 1.0.2

28.09.2025

**Добавлено**:

* Команды "Проверить на коллизии" и "Посмотреть отчет о коллизиях";
* В справку под именами команд добавлена информация, с какой версии они включены в плагин, также добавлен настоящий раздел с обновлениями в конец файла справки;

**Исправлено**

* Команда "Справка" - изменен механизм вызова приложения для открытия локального файла справки в PDF (раннее открывалась папка "Мои документы");
  
  # Версия 1.0.1

25.09.2025

**Исправлено**

* Команды "Вывести уклоны кровли": разблокированы поля настройки уклонов, переделана математическая логика. Теперь команда должна функционировать верно;

# Версия 1.0.0

23.09.2025

**Добавлено**:
