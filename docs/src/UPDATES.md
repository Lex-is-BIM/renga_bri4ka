# История обновлений плагина RengaBri4ka

# Версия 1.0.9

20.12.2025

**Добавлено**

- Команда "Командный препроцессор";

# Версия 1.0.8

18.12.2025

**Добавлено**

- Команда "Импорт рельефа";

**Изменено**

- Команда "Запрет выбора" - устранена ошибка вылета при попытке выбора объекта с его последующим удалением (процедура IModel.GetObjects() должна заключаться в try\catch);

# Версия 1.0.7

17.12.2025

**Добавлено**

- Команда "Запрет выбора";

**Изменено**

* Внутренние изменения в логике регистрации свойств;

* Команда "Видовой куб" - исправление подписи в параметрах ориентации камеры, окно будет закрываться после завершения работы Renga;

* Окно команды "Посмотреть отчет о коллизиях" теперь будет закрываться после закрытия Renga;

# Версия 1.0.6

03.11.2025

**Добавлено**

* Команда "Видовой куб";

**Изменено**

* Команда "Перекрытие по помещению" - теперь контур формируется корректно для любой геометрии перекрытия (раннее дуговые сегменты скашивались и пр.);

# Версия 1.0.5

11.10.2025

**Добавлено**

* Команды "Построить тени", "Перекрытие по помещению";

**Изменено**

* Команда "Проверить на коллизии" - добавлено кэширование обработанных объектов модели (при повторном вызове функции для данного проекта геометрия будет восстановлена из локального кэша вместо считывания из Renga), ускоряет скорость обработки;

* \[Internal\] Внутреннее изменение принципа хранения данных о вершинах граней солидов (объем ресурсов системы выделяет больше, чем раннее);

# Версия 1.0.4

29.09.2025

**Изменено**

* Внутренние исключения функций переопределены на вызов окна об ошибке Renga, чтобы не приводить к фатальному завершению работы в программе;
* Команда "Проверить на коллизии" - исправлен вывод в отчет коллизий с тэгом "\_Error" (появилось в версии 1.0.3);
* Из меню удалена команда "Справка", т.к. она не открывает нужный файл, см. спрвку в папке с плагином, файл "Bri4kaGuide.pdf";

# Версия 1.0.3

28.09.2025

**Добавлено**:

* Плагин теперь распространяется в двух редакция -- под `.NET Framework 4.8` и `.NET8`, подробнее см. [раздел об установке](./1.1_INSTALL.md);
* Команда "Проверить на коллизии" -- добавлена кнопка "Обратить выбор между группами" для обмена выбранными категориями между первой и второй группой объектов;
* Команда "Проверить на коллизии" -- добавлена опция "Анализировать только базовые линии (при наличии)" для возможности оптимизации анализа части объектов (те, геометрия которых может быть представлена в виде 3D-полилинии);
* Команда "Проверить на коллизии" -- добавлены опции настройки допуска при сравнении геометрии и частоту сегментации линейных сегментов (если стоит флажок "Анализировать базовые линии");

**Изменено**

* Команда "Проверить на коллизии" -- убран флажок о передаче параметра Id' коллизирующего объекта (эта функциональность доступа в общем случая для настройки передачи параметров);
* Команда "Посмотреть отчет о коллизиях" -- добавлено влияние галочек настройки выделения объектов при активном выборе объекта (раннее надо было повторять выбор позиции таблицы);

# Версия 1.0.2

28.09.2025

**Добавлено**:

* Команды "Проверить на коллизии" и "Посмотреть отчет о коллизиях";
* В справку под именами команд добавлена информация, с какой версии они включены в плагин, также добавлен настоящий раздел с обновлениями в конец файла справки;

**Изменено**

* Команда "Справка" - изменен механизм вызова приложения для открытия локального файла справки в PDF (раннее открывалась папка "Мои документы");
  
  # Версия 1.0.1
