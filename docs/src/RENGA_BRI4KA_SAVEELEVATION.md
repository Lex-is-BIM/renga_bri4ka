# Абсолютная отметка

*Доступно с версии 1.1.2*

Функция без каких-либо настроек. Рассчитывает для объектов, привязываемых к уровням, их абсолютную отметку (сумму отметки уровня и смещения относительно уровня), значение записывается в новое свойство типа Double "Bri4ka. Абсолютная отметка". Свойство будет иметься только у объектов, привязываемых к уровням.

Если отметка не была успешно получена, то свойство останется пустым.
