# Установка

На текущий момент плагин не имеет инсталлятора и на конечных ПК устанавливается вручную. Возможны 2 сценария установки и последующего обновления (зависит от наличия прав пользователя на папку с Renga):

- Вариант 1 (при отсутствии прав): единожды Администратором производится запись в папку Renga конфигурационного файла `.rndesk`, а все исполняемые библиотеки располагаются в ином доступном для редактирования каталоге;

- Вариант 2 (при наличии всех прав): файлы плагина распаковываются в папку Renga, при обновлении плагина файлы заменяются новыми;

Начиная с версии 1.0.3 плагин доступен в двух редакциях -- будучи собранный под `.NET Framework 4.8` (должен работать со всеми версиями без ограничений) и под .NET 8 (сможет работать только начиная с Renga 8.3). Версия под `.NET8` может использоваться на ПК с установленной Windows 11 (по каким-то причинам другая версия плагина, на .NET Framework 4.8, там может не запускаться).

Ниже приведена последовательность действий для установки для обоих вариантов. Развёртывание версии под `.NET Framework 4.8` и `.NET8` не отличается друг от друга, только используются файлы из соответствующих папок:

* `.NET Framework 4.8`: из папки `net48`;

* `.NET8`: из папки `net8.0-windows`;
1. Загрузите последнюю скомпилированную версию из раздела [Releases](https://github.com/Lex-is-BIM/renga_bri4ka/releases/latest)

2. Зайти в папку с установленной Renga нужной версии (папка, в корне которой лежит приложение `Renga.exe`);

3. Создать при отсутствии папку `Plugins`;

4. Создать в ней папку `RengaBri4ka`;

5. (при Варианте 1) Скопировать в папку `RengaBri4ka` файл `RengaBri4ka.rndesc` из распакованного архива, соответствующей подпапки `net48` или `net8.0-windows`. Открыть скопированный файл `RengaBri4ka.rndesc` и отредактировать XML-тэг `PluginFilename`, задав абсолютный файловый путь до `RengaBri4kaLoader.dll`, куда вы распаковали `RengaBri4ka` у себя на ПК

6. (при Варианте 2) Скопировать в папку `RengaBri4ka` все файлы из папки `net48` или `net8.0-windows`архива.

Если всё сделано корректно, то при запуске Renga на панели инструментов появится кнопка плагина, содержащего перечень функций на выпадающем списке;
