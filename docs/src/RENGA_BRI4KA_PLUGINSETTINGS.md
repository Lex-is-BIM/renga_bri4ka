# Настройки плагина

*Доступно с версии 1.1.1*

Модальное окно, содержащие различные настройки плагина

## Автозапуск функций

- "Задание стиля 3D-вила проекта" -- при каждом открытии или создании нового проекта на 3D-виде будет задаваться установленный визуальный стиль, отличный от монохромного. Настройки стиля по умолчанию приведены в элементе ниже. По умолчанию -- цветной;



Нажатие на кнопку "Применить изменения" задает настройки для данной сессии плагина и сохраняет их в файл в папке приложения. Окно будет автоматически закрыто
