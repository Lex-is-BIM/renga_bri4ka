# Связать помещение с перекрытием

*Доступно с версии 1.1.2*

Функция без каких-либо настроек.

Анализирует все помещения и перекрытия в модели. Если геометрический центр помещения попадает в контур перекрытия и отметки помещения с перекрытием совпадают, то у помещения заполняются свойства:

- "Bri4ka. Идентификатор перекрытия": значением будет уникальный идентификатор-Guid перекрытия (UniqueId);

- "Bri4ka. Наименование перекрытия": значением будет наименование перекрытия;
