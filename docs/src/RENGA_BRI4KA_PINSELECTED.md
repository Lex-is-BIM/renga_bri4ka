# Задать закрепление

*Доступно с версии 1.1.2*

Вызывает окно с перечислением всех категорий объектов в Renga:

![](assets/2026-01-30-22-07-57-image.png)

Необходимо выбрать одну или несколько категорий (отсортированных по алфавиту) и нажать "Сохранить выбор". Все объекты модели, относящиеся к указанным категориям будут Закреплены.
